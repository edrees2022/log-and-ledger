## الخطوات السريعة (5 دقائق)

1. **افتح Vercel**: https://vercel.com/dashboard
2. **اختر المشروع**: log-and-ledger
3. **اذهب إلى**: Settings → Environment Variables
4. **أضف المتغيرات**: انسخ من أعلى، واحد واحد
5. **حدد البيئات**: ✓ Production ✓ Preview ✓ Development
6. **أعد النشر**: Deployments → ⋯ → Redeploy

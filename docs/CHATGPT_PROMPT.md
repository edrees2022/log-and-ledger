أنا بحاجة لترجمة ملف JSON لتطبيق محاسبة من الإنجليزية إلى [**اللغة هنا**].

**القواعد المهمة جداً:**
1. ترجم فقط القيم (values) وليس المفاتيح (keys)
2. احتفظ بالهيكل JSON بالضبط كما هو - كل الأقواس والفواصل
3. لا تترجم: 
   - أسماء العلامات التجارية: Log & Ledger Pro
   - الاختصارات التقنية: API, PDF, URL, SMS, QR, GPS, USB, PIN, ATM, CEO, HR, IT, IBAN, VAT, EIN, SKU
   - Email, OK
4. استخدم مصطلحات محاسبية احترافية مستخدمة في [الدولة]
5. احتفظ بالأرقام والرموز كما هي
6. أرجع لي الملف JSON الكامل مترجم بنفس التنسيق
